#  Digital Clock using JavaScript

A simple digital clock built using **HTML, CSS, and JavaScript** that displays the current time in real-time and updates every second.

---

##  Features
- Shows current **hours, minutes, and seconds**
- Updates time automatically every second
- Clean and minimal UI
- Beginner-friendly code
- Uses JavaScript `Date` object and `setInterval`

---

##  Technologies Used
- HTML
- CSS
- JavaScript (Vanilla JS)
